\hypertarget{index_autotoc_md2}{}\doxysubsection{What is R-\/\+TYPE ?}\label{index_autotoc_md2}
\char`\"{}\+R-\/\+Type\char`\"{} is an iconic side-\/scrolling shoot them up video game series. This game is mainly about defeating enemies, surviving difficult levels, collecting power-\/ups to improve your ship, and progressing through the different levels of this adventure. You can find more details regarding this game through the following resource \href{http://www.hardcoregaming101.net/r-type/}{\texttt{ click on me !}}\hypertarget{index_autotoc_md3}{}\doxysubsection{Project description}\label{index_autotoc_md3}
This project aims to implement a multithreaded server and a graphics client for an existing video game well known to everyone, the \char`\"{}\+R-\/\+TYPE\char`\"{}. In other themes, it is about setting up in C++ our own r-\/type game but with additional requirements not present in the original version of the game. It is indeed about being able to achieve\+:
\begin{DoxyItemize}
\item \mbox{[}\mbox{]} A network game in which one to four players can fight the evil Bydos together and
\item \mbox{[}\mbox{]} Architectural models of a real game engine.
